# Обзор литературы и существующих решений

## 1. Введение

В последние годы наблюдается активное развитие систем генеративного искусственного интеллекта, применяемых в задачах дизайна интерьеров. Такие системы позволяют автоматизировать процесс создания визуальных концепций помещений, снижая порог входа для пользователей без профессионального опыта.
Современные решения используют комбинацию:
- больших языковых моделей (LLM),
- генеративных моделей изображений (diffusion models),
- а также инструментов 3D-визуализации и AR/VR.

AI-системы в дизайне интерьеров позволяют генерировать концепции, вариации стилей и визуализации на основе текста или изображений, ускоряя процесс проектирования и принятия решений.

## 2. Классы существующих решений

### 2.1 Генерация изображений по тексту (Text-to-Image)

К данному классу относятся модели, генерирующие изображения интерьеров по текстовому описанию:
- Midjourney
- DALL-E
- Stable Diffusion

Такие системы позволяют описать интерьер словами и получить визуализацию. Они хорошо подходят для генерации идей и концептов.

Особенности:

- высокая визуальная реалистичность
- гибкость за счёт текстового управления
- зависимость от качества промпта

Сравнение показывает, что Midjourney лучше передаёт атмосферу и стиль, а DALL-E лучше следует точным инструкциям.

Недостатки:

- сложность управления результатом
- необходимость навыков prompt-инжиниринга
- отсутствие структурированного ввода

### 2.2 AI-платформы для проектирования интерьеров

К этому классу относятся более комплексные решения:

- Planner 5D
- Homestyler
- Roomstyler
- Foyr Neo

Они позволяют:

- создавать планировки
- расставлять мебель
- работать с 3D-визуализацией

Например, Planner 5D использует AI для автоматического размещения мебели и генерации реалистичных интерьеров.

Преимущества:

- высокая точность
- работа с реальными размерами
- визуализация в 3D

Недостатки:

- сложный интерфейс
- избыточность для обычного пользователя
- необходимость ручной настройки
- стоимость

### 2.3 Генерация дизайна по фото (Image-to-Image)

Отдельный класс решений — редизайн интерьера по фотографии:

- RoomGPT
- ReimagineHome
- IA Decora

Принцип работы:

- пользователь загружает фото комнаты
- система предлагает варианты редизайна

Например, RoomGPT генерирует несколько вариантов дизайна по одному изображению (Tools and Hands).

Преимущества:

- высокая персонализация
- удобство для пользователя

Недостатки:

- ограниченный контроль
- шаблонность решений
- слабая точность деталей

## 3. Научные исследования

В научной литературе активно развиваются методы генерации интерьеров с использованием ИИ.

Основные направления:

- Генерация по тексту и изображению
    - системы, объединяющие текстовые и визуальные входные данные повышают точность и согласованность результата
    - пример: система VIDES генерирует интерьеры по тексту и визуальным подсказкам, улучшая качество и соответствие ожиданиям пользователя

- Оптимизация планировки
    - использование LLM для генерации структуры помещения
    - пример: LLM может извлекать ограничения и оптимизировать размещение мебели

- Интерактивные системы
    - пользователь взаимодействует с моделью в процессе генерации
    - пример: Chat2Layout позволяет управлять расстановкой мебели через диалог с моделью (arXiv)

- Контролируемая генерация
    - улучшение управляемости diffusion-моделей
    - пример: DiffDesign вводит контроль параметров (стиль, размер, композиция) (arXiv)

## 4. Ограничения существующих решений

Несмотря на развитие технологий, большинство решений имеют ряд ограничений:
- сложность управления результатом
- зависимость от качества текстового запроса
- недостаток структурированных параметров
- ограниченная персонализация
- иногда нереалистичные или шаблонные результаты

Также часто отсутствует баланс между простотой использования и точностью управления

## 5. Выводы и позиционирование проекта

Анализ существующих решений показывает:
- Генераторы изображений дают красивый результат, но плохо управляются
- Профессиональные инструменты точны, но сложны
- Photo-based решения удобны, но ограничены

Таким образом, существует ниша для решения, которое сочетает простоту интерфейса и структурированное управление параметрами; использует LLM для улучшения промпта и генерирует реалистичные изображения

## 6. Отличие нашего решения
Разрабатываемый MVP занимает промежуточную позицию: проще профессиональных инструментов и более управляемый, чем чистые text-to-image модели

Ключевые особенности:
- использование LLM для генерации промпта
- добавление структурированных параметров (стиль, размер, освещение)
- простой пользовательский интерфейс

Таким образом, проект решает одну из ключевых проблем существующих решений — недостаток управляемости при сохранении простоты использования.
